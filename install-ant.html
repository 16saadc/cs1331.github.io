Installing Ant

Mac

If you have homebrew, try: brew install ant
To install homebrew: ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Windows/Linux

  1. Download Ant from the Ant Download Page
  2. Follow the installation guide carefully.

Confirm Installation

ant -version should return something like: Apache Ant(TM) version 1.9.4 compiled on {date}
